# Proposal: Improve `synck create` quality

## Goal

Upgrade `synck create` to scaffold higher-quality skills by default.

## Scope

- Generate full pack: `SKILL.md`, `assets/SKILL-TEMPLATE.md`, `references/README.md`, `CHECKLIST.md`
- Enforce strict prompt validation
- Enforce no-overwrite behavior for existing skill directories

## Non-goals

- AI-generated domain-specific examples
- Automatic AGENTS.md registration in create step
